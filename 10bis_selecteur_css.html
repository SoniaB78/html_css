Propriétés CSS utilisées dans cette page

Décrivez leur usage et les valeurs possibles en fonction de ce que vous avez déjà vu

  1. background : on peut l'utiliser pour la couleur de fond, il peut prendre plusieurs valeur. (reapeat, url(''), fixed ...)
  2. color : change la couleur du texte peut prendre un nom (en anglais), rgb ou un hexadécimal ...
  3. float : retir du flux un élément peut prendre les valeurs right et left.
  4. font-family : police du texte, exemple times new roman.
  5. font-size : permet d'augmenter ou de baisser la taille du texte exemple font-size: 20px (px, %, em, rem)
  6. height : règle la hauteur, (px, %, vh, vw)
  7. left : pour mettre à gauche (px, %, ...), fonctionne avec un propriété position
  8. line-height : espacement entre les lignes de texte(interligne) (px, %, em, rem)
  9. margin : taille des marge exterieurs, peut prendre jusqu'à 4 valeurs, sens horaire (px, %, em ...)
  10. padding : définit la taille des marges intérieur, sens horaire (px , ...)
  11. position : permet de changer la position d'une balise, statique, relative, fixed ...
  12. text-align : permet d'aligner un texte, center, justify, right, left
  13. text-decoration : permet de retirer le soulignement avec la valeur none
  14. top : fonctionne avec position, permet de situer sur l'axe des ordonnées la position d'une balise
  15. transition : précise un paramètre d'une animation comme sa durée, peut prendre une valeur en secondes.
  16. width : permet de modifier la largeur d'un balise, sa valeur est comme height

Propriétés CSS à rechercher

Recherchez les propriétés ci-dessous, faites des tests avec cette page depuis la console de debuggage (F12 sur Firefox ou clic droit "inspecter").

  1. font-weight : change l'état d'une police d'écriture -> graisse (bold, valeur de 1 à 1000).
  2. letter-spacing : change l'espace entre les caractères, px, em, rem.
  3. word-spacing : détermine l'espace entre les mots px, em, ...
  4. text-shadow : ombrage du texte, en px, x, y fondu, couleur.
  5. backgound-image : modifie l'arrière plan d'une balise, prend une url, un dégradé comme valeurs
  6. background-size : défini la taille du background, cover, px %, contain
  7. opacity : défini la transparence de la balise, 0 à 1, 0 -> invisible
  8. boder-style : défini le style de bordure utilisé, dotted, dashed, solid, double, groove, ridge
  9. box-shadow : ajoute une ombre derrière une balise en px %, x, y, blur, diffusion black
  10. list-style-image : remplace la puce d'une liste par une image, url('')
  11. border : ajoute une bordure, couleur, taille en px, style de trait
  12. border-radius : arrondie le coin d'un élément, sa valeur est en px, %, em...
  13. border-width : défini la largeur d'une bordure en px, em et ex
  14. font-variant : modifie le style de la police, pluisieurs valeurs mall-caps, normal ...
  15. background-color : caractérise la couleur du background, hexadécimal, couleur en anglais ...
  16. !important : à mettre à la fin d'une ligne, force le navigateur à utiliser ce style
